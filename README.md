# 🍴 Foodie Fiesta

A simple, clean, and responsive food ordering website created using HTML and CSS.

## 🌟 What’s Inside?
* **Home Page**: A welcoming landing page with a clear description and an "Order Now" button.
* **About Us**: A creative section that showcases our team of chefs with smooth hover animations.
* **Menu**: A modern grid layout where you can see our specialties like Pizza, Burgers, and Pasta.
* **Order Now**: A simple booking form to collect your name, phone number, and address.
* **Header & Footer**: Consistent navigation on every page so you never get lost.

## 🛠️ Built With
* **HTML5**: To build the structure of the pages.
* **CSS3**: To make the website look professional and colorful with custom animations.
* **JavaScript**: A tiny bit of code to show a "Thank You" message after you order.

## 🚀 How to Use It
1. Download all the files.
2. Make sure the images are in a folder named `image`.
3. Open `index.html` in any web browser to view the site!
